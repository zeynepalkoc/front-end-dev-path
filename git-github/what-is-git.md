# What is Git?

Git is a distributed version control system.

It helps developers track changes in their code and collaborate with others.

Git allows developers to:

- track code history
- revert to previous versions
- collaborate with teams
- manage project versions

Git is widely used in modern software development.

---

# Why Git is Important

In software development, code changes frequently.

Without version control:

- changes can be lost
- collaboration becomes difficult
- tracking bugs becomes harder

Git solves these problems.

---

# What is Version Control?

Version control is a system that records changes to files over time.

Example:

Version 1 → initial project  
Version 2 → added login feature  
Version 3 → fixed bug  

Git keeps track of all these versions.

---

# Git vs GitHub

Git and GitHub are not the same.

Git

- version control system
- runs locally on your computer

GitHub

- cloud platform
- stores Git repositories online
- enables collaboration

---

# Example Workflow

Typical developer workflow:

1. Write code
2. Track changes with Git
3. Push code to GitHub
4. Collaborate with others

---

# Example

A developer creates a project.
