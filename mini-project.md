# HTML Mini Project: Personal Profile Page

In this mini project, we will create a simple personal profile webpage using the HTML concepts learned in this module.

Concepts used in this project:

- headings
- paragraphs
- images
- links
- lists
- semantic HTML
- basic page structure

---

# Project Goal

Create a simple personal webpage that includes:

- a header
- navigation links
- an about section
- a skills list
- a contact link
- a footer

---

# Example Project Structure

```html
<!DOCTYPE html>
<html lang="en">

<head>
<meta charset="UTF-8">
<title>My Profile</title>
</head>

<body>

<header>
<h1>My Personal Website</h1>
</header>

<nav>
<a href="#">Home</a>
<a href="#">About</a>
<a href="#">Contact</a>
</nav>

<section>

<h2>About Me</h2>

<p>Hello! My name is Zeynep.</p>
<p>I am learning frontend development.</p>

<img src="profile.jpg" alt="Profile picture" width="200">

</section>

<section>

<h2>My Skills</h2>

<ul>
<li>HTML</li>
<li>CSS</li>
<li>JavaScript</li>
</ul>

</section>

<footer>

<p>Contact: example@email.com</p>

</footer>

</body>

</html>
