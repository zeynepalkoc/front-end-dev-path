# What is React?

React is a JavaScript library used to build user interfaces.

It was developed by Facebook and is widely used for creating modern web applications.

React allows developers to build **interactive and reusable UI components**.

---

# Why React is Popular

React is popular because it:

- uses reusable components
- improves performance with Virtual DOM
- has a large community
- works well with modern frontend tools

React is used by companies such as:

- Facebook
- Netflix
- Instagram
- Airbnb

---

React applications are built using components.

A component is a reusable piece of UI.

Virtual DOM

React uses a Virtual DOM to improve performance.

Instead of updating the whole page, React updates only the necessary parts.

This makes applications faster and more efficient.
