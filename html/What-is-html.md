# What is HTML?

HTML stands for **HyperText Markup Language**.

It is the standard language used to create the structure of web pages.

HTML is used to organize content such as:

- headings
- paragraphs
- links
- images
- lists
- forms

HTML is not used for styling or interactivity.

- **HTML** gives structure
- **CSS** gives style
- **JavaScript** gives behavior

## Why is HTML important?

HTML is the foundation of every website.

Without HTML, a browser would not know what to display on a page.

## Basic Example

```html
<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello World</h1>
    <p>This is my first web page.</p>
  </body>
</html>

Summary

HTML is the basic building block of the web.
It helps browsers understand and display page content.
