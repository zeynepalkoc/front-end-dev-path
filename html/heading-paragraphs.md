# Headings and Paragraphs in HTML

Headings and paragraphs are used to structure text content on a web page.

They help organize information and improve readability.

---

# HTML Headings

HTML provides **six levels of headings**.

They range from `<h1>` to `<h6>`.

Example:

```html
<h1>Main Title</h1>
<h2>Section Title</h2>
<h3>Subsection</h3>
<h4>Small Heading</h4>
<h5>Smaller Heading</h5>
<h6>Smallest Heading</h6>
```
